Cheat sheet ligne de commande

Système de fichiers :

  • cd = Se déplacer dans un autre répertoire.
  • df = Indiquer l'espace occupé par les systèmes de fichiers.
  • du = Evaluer l'espace disque occupé par des fichiers.
  • file = Déterminer le type d'un fichier.
  • stat = Information détaillées sur un fichier.
  • ls = Lister les fichiers et dossiers présent dans un répertoire.
  • pwd = Afficher le chemin vers le répertoire de travail courant.

Manipulation :

  • cp = Copie.
  • mv = Déplacement.
  • rm = Suppression.
  • mkdir = Création de répertoire.
  • rmdir = Suppression de répertoire.

Recherche :

  • find = Rechercher des fichiers.
  • grep = Faire des recherches plein texte.
  • locate = Lister les fchiers a partir d'une base de données.
  • wc = Compter les lignes,mots...

Installer des logiciels :

  • apt-get = Installer et désinstaller des paquets.
  • apt-cache = Faire des recherches dans le cache d'APT et affichers des informations sur les paquets.
  • dpkg = Gestion de paquet de niveau avancé.
  • apt-key = Gérer les clès de sécurité des dêpots de paquets.
